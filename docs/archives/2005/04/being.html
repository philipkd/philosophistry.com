I just want to be.

I don't want to sleep
I don't want to walk

I don't want to eat
I don't want to think

I don't want to talk
I don't want to cry

I don't want to read
I don't want to fight

I don't want to listen
I don't want to move

I don't want to meditate
I don't want to eat

I don't want to play
I don't want to go

I don't want to surf the Internet
I don't want to blog

I don't want to pause
I don't want to paint

I just want to be. right here, right now, not relaxing, not un-relaxing, not forcing a peace, not silencing myself, but just being.
